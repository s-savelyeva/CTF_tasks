## Описание

Перед решением этой задачи, нужно решить первую часть «Ангара-1: Вход в ОС».

Отлично, мы в системе! Теперь необходимо найти коды запуска ракеты.

Флаг для сдачи: BugCTF{код_запуска}

## Решение

Заходим с помощью пароля из прошлого задания в аккаунт, там скачиваем картинку

Картинку через zsteg анализируем и находим строку:

```
zsteg sochi-2020.png

b1,rgb,lsb,xy       .. text: "The password to the archive is in the rockyou.txt database"
```

Скачиваем архив doc.zip с сайта и вскрываем его брутфорсом:

```
fcrackzip -u -D -p rockyou.txt doc.zip

PASSWORD FOUND!!!!: pw == l0v3
```

Открываем архив, переходим по ссылке в текстовом документе, флаг скрыт невидимыми символами в тексте:
