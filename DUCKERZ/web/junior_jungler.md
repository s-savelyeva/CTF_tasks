## Описание

Вам необходимо научиться самому важному трюку жонглера. Попадите на его представление и узнайте все секреты.

## Решение

В PHP при использовании ==:
```
Строка, начинающаяся с "0e" и содержащая только цифры после, интерпретируется как научная нотация

"0e123" считается равным 0 × 10¹²³ = 0

"0e974114726967211511915142978119" == 0
```
Подбираем хэш с 0e:
```
240610708 → md5 = 0e462097431906509019562988736854
```
Почему это работает:
```
md5("240610708") = "0e462097431906509019562988736854"

PHP видит: "0e462097431906509019562988736854" == "0e974114726967211511915142978119"

Обе строки интерпретируются как 0 (0 × 10^... = 0)

0 == 0 → TRUE
```
