## Описание

Хотел открыть изображение, а оно не открывается. Поможешь мне?

## Решение

Открываем файл через 010editor, видим, что неправильно заполнены первые байты для png изображения:
```
00 00 4E 47 0D 0A 1A 0A 00 00 00 0D 49 48 44 52
```
А должно быть:
```
89 50 4E 47 0D 0A 1A 0A
```

Заменяем и получаем флаг в изображении:

DUCKERZ{HEXr3d4ct0r_v3ry_us3full}
